## 90-Day Execution Calendar

### Days 1-14
- Lock measurement taxonomy
- **Run CWV baseline audit on all public pages** (moved from Phase 06)
- Build content registry architecture
- Design `/guides` and `/compare` route system
- Audit all current public pages
- Create content inventory + keyword map
- **Set up `/llms.txt` for AI Search readiness**

### Days 15-30
- Ship guides hub + detail templates
- Migrate existing support pages
- Harden money pages with stronger internal links and FAQs
- **Implement internal link architecture (hub-spoke model)**
- **Design + deploy unique OG images for 5 money pages**
- Registry-driven sitemap start
- canonical/hreflang audit pass
- **Implement schema strategy matrix across page types**

### Days 31-45
- Publish first wave:
  - 6-8 guides
  - 3 comparison pages
- Add page-level analytics and weekly dashboard
- Create first refresh backlog

### Days 46-60
- Publish second wave:
  - 6-8 more guides (including device-specific pages with ≥40% unique content)
  - 2-3 more comparisons
- Expand trust/entity pages
- Improve CWV and rendering for public pages (fix issues from day 1-14 baseline)
- **Set up YouTube channel + upload initial demo videos**
- **Add VideoObject schema to pages with embedded videos**
- **Create video sitemap**

### Days 61-75
- Review GSC data
- CTR iteration on homepage + top money pages
- **Featured snippet optimization on top-ranking guides**
- Consolidate overlapping pages
- Start authority/distribution outreach (Tier 1 directories + brand profiles)
- **Submit to Product Hunt, AlternativeTo, Slant**

### Days 76-90
- Publish third wave only if first two waves show quality
- Decide locale readiness
- Finalize operational dashboards
- Lock quarterly refresh/prune cadence

## 12-Month Operating Cadence

### Month 1-3
- Build system
- publish initial EN cluster
- stabilize data and templates

### Month 4-6
- scale EN content
- deepen comparison cluster
- start authority layer
- maybe launch VI pilot

### Month 7-9
- expand proven locale only
- run content refresh and consolidation
- mature brand/entity signals

### Month 10-12
- optimize winners
- prune losers
- expand only into clusters proven to convert
- harden long-term editorial + SEO governance

## Weekly Rituals
- GSC review
- page winner / loser review
- new topic candidate selection
- content QA

## Monthly Rituals
- scorecard review
- CTR experiments
- refresh queue
- authority progress review

## Quarterly Rituals
- consolidation/pruning audit
- locale expansion decision
- architecture debt review
- content ownership reset

## Team Shape Needed if Fully Maximized
- SEO strategist
- technical SEO / frontend implementer
- content lead
- editor
- analyst
- outreach / distribution owner

## Success Criteria
- Team operates SEO as a system, not a burst of page publishing.

## Risk Assessment
- Publishing outruns QA.
- Ops load becomes chaotic without ownership.

## Security Considerations
- Do not let content production pipelines bypass review.
